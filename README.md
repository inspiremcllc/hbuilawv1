# H Bui Law Website Mockup

Deploy-ready static Next.js mockup for a Southern California family law firm pitch.

## Local Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run lint
npm run build
```

## Vercel Deployment Notes

Import this folder into Vercel as a Next.js project. No database, backend route, or paid API key is required. The contact form is static and shows a local mockup success message only.

## Mockup Notes

This site is for demonstration purposes only. Placeholder attorney details, location addresses, testimonials, map blocks, and payment processor copy should be reviewed and replaced before any live client launch.
